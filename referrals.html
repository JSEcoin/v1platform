The platform is available to users from any country but only new users from these countries will earn referral payments. This is because we are using these countries as primary target regions while the system is still being built. We currently have a translations module in development and we will likely expand the affiliate program to more countries in the future.

Please note that affiliates can be based in any country and still receive referral earnings. The restriction is on the geoip location of the new user that has been referred.
